# /create-feature - Create New Feature with TDD

Implement new feature using test-driven development workflow.

## Usage

```bash
/create-feature <feature-name>

# Examples:
/create-feature user-authentication
/create-feature api-rate-limiting
```

## Workflow

1. **Research** - Understand requirements
2. **Plan** - Design implementation
3. **TDD Implement** - Write test → Code → Refactor
4. **Validate** - Run full test suite
