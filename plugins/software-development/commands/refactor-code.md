# /refactor-code - Refactor Code Safely

Refactor code while maintaining test coverage and behavior.

## Usage

```bash
/refactor-code <target>

# Examples:
/refactor-code auth/handler.py
/refactor-code src/api/
```

## Workflow

1. **Research** - Understand current code
2. **Plan** - Design refactoring steps
3. **Implement** - Refactor with tests passing
4. **Validate** - Ensure no behavior change
