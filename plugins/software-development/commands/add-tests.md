# /add-tests - Add Tests to Existing Code

Improve test coverage for existing code.

## Usage

```bash
/add-tests <target>

# Examples:
/add-tests src/utils.py
/add-tests src/api/
```

## Workflow

1. **Research** - Analyze code and current coverage
2. **Plan** - Identify untested paths
3. **Implement** - Write tests for gaps
4. **Validate** - Check coverage improved
